# ダウンロード＆インストール

Kiro IDEのインストールガイドです。

## Kiroをダウンロード

1. [kiro.dev](https://kiro.dev)にアクセス
2. お使いのOS用のインストーラーをダウンロード
3. OS固有のセットアップ手順に従ってインストール（Windows、macOS、またはLinux）

## 初回起動

初回起動時に以下の手順を実行します：

1. **認証**: ソーシャルログインまたはAWSログインで認証
2. **VS Code設定のインポート**（オプション）: 既存のVS Code設定をインポート可能
3. **テーマの選択**: お好みのテーマを選択
4. **シェル統合の有効化**: ターミナル統合を設定
5. **最初のプロジェクトの作成**: セットアップ完了後、プロジェクトを開始

## 言語サポート

- **TypeScript/JavaScript**: [TypeScript/JavaScriptガイド](../guides/languages-and-frameworks/typescript-javascript.md)
- **Python**: [Pythonガイド](../guides/languages-and-frameworks/python.md)

各言語の環境設定について詳しくはそれぞれのガイドを参照してください。
