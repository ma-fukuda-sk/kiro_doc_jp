# Kiro CLI

Kiro CLIは、AI支援開発をターミナルにもたらすコマンドラインインターフェースです。

## インストール

macOSおよびLinuxでのインストール：

```bash
curl -fsSL https://cli.kiro.dev/install | bash
```

詳細は[インストールガイド](./installation.md)を参照してください。

## コア機能

### 1. Interactive Chat（インタラクティブチャット）

ターミナルで自然な会話を通じてKiroと対話

### 2. Custom Agents（カスタムエージェント）

ワークフロー向けの専門エージェントを作成・デプロイ

### 3. MCP Integration（MCP統合）

Model Context Protocolを通じて外部ツールに接続

### 4. Smart Hooks（スマートフック）

インテリジェントなpre/postコマンドフックでワークフローを自動化

### 5. Agent Steering（エージェントステアリング）

ベストプラクティスを使用してエージェントの動作をガイド

### 6. Auto Complete（オートコンプリート）

コンテキスト認識型コマンド補完

## 主なユースケース

- ターミナルベースのインタラクティブ開発支援
- カスタムエージェントによるワークフロー自動化
- チームレベルのプラクティス標準化
- サードパーティツール統合
- コンテキスト認識型コーディング提案
- 繰り返しタスクの自動化
