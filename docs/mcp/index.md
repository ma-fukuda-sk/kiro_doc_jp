# MCP（Model Context Protocol）

MCPは、Kiroが外部サーバーと通信して専門的なツールや情報にアクセスすることを可能にします。

## MCPとは？

MCPは以下を可能にします：

- ナレッジベースへのアクセス
- 外部サービスとの統合
- ドメイン固有の機能でKiroを拡張

## MCPのセットアップ

### 前提条件

- 最新バージョンのKiroがインストールされていること
- サーバー固有の前提条件

### MCPサポートの有効化

1. Settings を開く（`Cmd+,`（Mac）または `Ctrl+,`（Windows/Linux））
2. MCPサポートを有効化

### MCPサーバータブの使用

MCPサーバータブには以下が表示されます：

- 設定済みサーバー
- 接続ステータス
- ツールアクセス

チャットでツール名をクリックしてプレースホルダープロンプトを挿入できます。

## トラブルシューティング

### MCPログの確認

OutputタブでKiro - MCP Logsを確認できます。

### よくある問題と解決策

- 接続失敗
- パーミッションエラー
- 設定の問題
