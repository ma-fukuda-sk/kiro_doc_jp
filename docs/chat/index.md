# Agentic Chat

Kiroは、自然言語会話を通じてコードと対話できるチャットパネルを提供します。

## 主な機能

- **Contextual Understanding（コンテキスト理解）**: コードベース分析
- **Smart Intent Detection（スマート意図検出）**: ユーザーのニーズを理解
- **Vibe vs Spec sessions**: セッションタイプの比較
- **Dev Servers**: バックグラウンドプロセス

## はじめに

### チャットへのアクセス

以下の方法でチャットにアクセスできます：

- **キーボードショートカット**: `Cmd+L`（Mac）/ `Ctrl+L`（Windows/Linux）
- **コマンドパレット**: コマンドを入力
- **サイドバー**: チャットアイコンをクリック

### 最初の会話

例えばリクエストの例：
- コードの説明を求める
- バグの修正を依頼
- 新機能の実装を依頼

### 会話のエクスポート

会話をマークダウン形式でエクスポートできます。

### スマート意図検出

Kiroはあなたのリクエストを理解し、適切なアクションを提案します。

## コンテキスト管理

チャットで`#`シンボルを使用してコンテキストプロバイダーにアクセスできます。

### 利用可能なコンテキストプロバイダー

| プロバイダー | 説明 |
|-------------|------|
| `#codebase` | コードベース全体を参照 |
| `#file` | 特定のファイルを参照 |
| `#folder` | フォルダを参照 |
| `#git` | Git情報を参照 |
| `#terminal` | ターミナル出力を参照 |

**ヒント**: ターミナル統合はデバッグに特に強力です。

## セッションと履歴

### セッションの作成と切り替え

複数のセッションを作成し、切り替えることができます。

### タスク追跡

タスクの進捗を追跡できます。

### 履歴の表示

検索可能なセッションログで実行履歴を確認できます。
