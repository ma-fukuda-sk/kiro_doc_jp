# Kiro ドキュメント

Kiroは、specs、steering、hooksなどの機能を備えたエージェント型IDEで、最高の仕事をサポートします。

## はじめに

Kiroを5分以内で始めるために必要なすべての情報。

- [ダウンロード＆インストール](./getting-started/installation.md)
- [最初のプロジェクト](./getting-started/first-project.md)

## コア機能

- **[Specs](./specs/index.md)**: 構造化された仕様を使用して機能を計画・構築
- **[Hooks](./hooks/index.md)**: インテリジェントなトリガーで繰り返しタスクを自動化
- **[Agentic Chat](./chat/index.md)**: AIとの自然な会話で機能を構築
- **[Steering](./steering/index.md)**: カスタムルールとコンテキストでAIをガイド
- **[MCP Servers](./mcp/index.md)**: 外部ツールやデータソースに接続
- **[プライバシーとセキュリティ](./privacy-and-security/index.md)**: プライバシーコントロールでコードを安全に保護
