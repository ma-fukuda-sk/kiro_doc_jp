# Management（管理）

Kiro IDEでのhook管理に関する重要なテクニックを説明します。

## 概要

効果的なhook管理により、プロジェクトの成長に伴って自動化ワークフローを整理し、保守可能で効率的に維持できます。すべてのhookコントロールはKiroのメインパネルのAgent Hooksセクションからアクセスできます。

## コア機能

### 有効化/無効化

hookを削除せずにオン/オフを切り替えられます。

- Agent Hooksパネルの目のアイコンをクリック
- またはhookインターフェース内のHook Enabledスイッチを使用

### 編集

既存のhookを選択して以下を調整できます：

- トリガー
- ファイルパターン
- 指示
- 説明

変更は即座に適用されます。

### 削除

Delete Hookオプションでhookを完全に削除できます。

**警告**: この操作は元に戻せません。

### 手動実行

Manual triggerのhookは2つの方法で実行できます：

- パネル内の再生ボタン
- hookビュー内のStart Hookボタン
