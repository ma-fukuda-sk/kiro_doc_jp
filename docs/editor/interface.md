# Kiro Interface

Kiro IDEインターフェースのナビゲーションと使用に関する包括的なガイドです。

## メインインターフェースコンポーネント

Kiroのインターフェースは5つの主要な要素で構成されています：

### 1. Editor（エディタ）

中央のワークスペースで以下の機能を提供：

- シンタックスハイライト
- 行番号
- コード折りたたみ
- 複数タブ
- 分割ビュー編集

### 2. Chat Panel（チャットパネル）

AI対話ツールで以下が可能：

- コードに関する質問
- コード生成リクエスト
- デバッグ支援
- コードレビュー

### 3. Views（ビュー）

専門パネルを含むサイドバー：

- **Explorer**: ファイルエクスプローラー
- **Search**: 検索
- **Source Control**: ソース管理
- **Run and Debug**: 実行とデバッグ
- **Extensions**: 拡張機能
- **Kiro固有の機能**: Specs、Hooks、Steering、MCP

### 4. Status Bar（ステータスバー）

下部の情報表示：

- ファイル詳細
- Gitステータス
- エラー数
- エージェントインジケーター

### 5. Command Palette（コマンドパレット）

クイックアクセスツール：

- ショートカット: `Cmd+Shift+P`（Mac）または `Ctrl+Shift+P`（Windows/Linux）
- アクションの実行
- MCPツールへのアクセス

## Chat Panel機能

- コード修正のリクエスト
- デバッグに関する質問
- `#`コマンドによるコンテキストのインクルード
- ボイラープレートコードの生成

## Kiro Sidebarビュー

Kiroサイドバービューは以下へのアクセスを提供：

- Specs管理
- Agent Hooks
- Agent Steering設定
- MCPサーバー

## ナビゲーションのヒント

- キーボードショートカットを使用
- コマンドパレットを活用
- よく使うファイルをピン留め
- コード比較に分割ビューを活用
- ワークスペース設定を構成
