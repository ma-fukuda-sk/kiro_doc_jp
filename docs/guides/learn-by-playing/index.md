# Learn by Playing（ゲームで学ぶ）

このガイドでは、Spirit of Kiroビデオゲームのコードベースでタスクを完了しながらKiroの使用方法を学びます。Spirit of Kiroのコードの95%はKiroへのプロンプトによって書かれています。

## ゲーム説明

Spirit of Kiroは無限クラフトゲームで、プレイヤーは以下を行います：

1. ユニークに生成されたオブジェクトを発見
2. シミュレートされたインタラクション（切る、塗る、接着、エンチャントなど）を通じてアイテムを互いに使用し、組み合わせ結果を生成
3. AI鑑定士に作品を販売

すべてのオブジェクトとインタラクションはAI生成され、無限のリプレイ性を提供します。

## 学習パス構造

ガイドは9つの連続タスクで構成されています：

### 1. Setup（セットアップ）

環境設定とローカルデプロイメント

### 2. Improve Homepage（ホームページの改善）

プロジェクトコンテキストのためのsteeringファイルの使用

### 3. Physics Bug Fix（物理バグ修正）

タブ切り替えグリッチのデバッグ

### 4. Interactions Bug（インタラクションバグ）

AI生成インタラクションロジックの修正

### 5. Code Refactoring（コードリファクタリング）

DRY原則の適用

### 6. Complex Feature（複雑な機能）

メール認証とパスワードリセットの実装

### 7. Asset Automation（アセット自動化）

ボイラープレート削減のためのagent hooks

### 8. MCP Extension（MCP拡張）

Model Context ProtocolでKiroを拡張

### 9. Conclusion（結論）

学習の振り返り
